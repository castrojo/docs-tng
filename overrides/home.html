Highlights

  • Comprehensive support for public cloud services and resources with a rich library of actions and filters to build policies with
  • Arbitrary filtering on resources with nested boolean conditions.
  • Automatically provisions serverless functions and event sources (AWS CloudWatchEvents, AWS Config Rules, Azure EventGrid, GCP AuditLog & Pub/Sub, etc)
  • Robust dry-run features, experiment with confidence
  • Policy-based outputs sent directly to cloud provider native metrics
  • Policy-based structured outputs of matched resources into cloud native object storage
  • Intelligent cache usage to minimize api calls.
  • Supports multi-account, subscription, project usage.
  • Battle-tested since 2015 - in production at some of the largest deployments on cloud
  • Robust and open community - Friendly and professional community dedicated to helping move Code as Governance forward. Part of the CNCF Ecosystem of tools (CNCF Sandbox)

The Path to a Well Managed Cloud

Cloud Custodian enables users to be well managed in the cloud. The simple YAML DSL allows you to easily define rules to enable a well-managed cloud infrastructure, that's both secure and cost optimized. It consolidates many of the ad-hoc scripts organizations have into a lightweight and flexible tool, with unified metrics and reporting. Custodian supports managing AWS, Azure, and GCP public cloud environments.
